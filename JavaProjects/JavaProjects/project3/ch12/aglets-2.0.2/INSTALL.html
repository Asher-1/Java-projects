Aglets SDK Install

Installation Instructions

  1. Change to the bin directory.
  2. Run the command:  ant
  3. Follow the displayed instructions for installing a keystore and policy file.  Look for the *** READ ME *** 's.

To run the server with the default configuration file:

  • NT:    agletsd -f ..\cnf\aglets.props
  • Unix: ./agletsd -f ../cnf/aglets.props
